2018二月回顾

Posted on 2018-03-04 | In 杂谈

在你浑浑噩噩的时候，时间从来不会等你，它只会默然前行。

2018 年已经过去了六分之一了，2017 年年底定的计划又完成了多少呢？是时候回顾这两个月，看看这两个月过的如何，是否有按计划对自己做出一些改变了。

  • 英语：每天单词都有在背；一天 100 个单词打卡，这个事情基本上已经养成了习惯，即使节假日也不会中断，这是一件好事；同时希望每天开始看两篇英语的文章，从简单的背单词开始想看文章迈进；

  • 健身：一月底的时候健身基本每周两三次的频率，二月之后由于频临春节，导致去健身的频率下跌，有时候一周只有一次；二月下旬回来之后开始保持每周两三次的频率，之后搬家了可能会提高去健身的频率，感觉健身虽然很累但是自己会很有成就感；

  • 看书：春节后回来至今基本上没有看过书，一个是因为想放松；一个是由于迷茫不知道到底要看哪个类型的书籍，加上在家一直有玩游戏，回来之后也没有戒掉，这些都是问题；基本上接下来的看书计划首先是把《鸟哥的linux私房菜》看完，然后看一本非技术型文章，争取三月份看完；

  • 博客：春节期间至今还是断了，既定计划没有完成；接下来应该多看书，多思考，争取两周能出来一篇中等质量偏上的文章就谢天谢地了，目前博客依然没有指定主题，可以写编程相关，也可以写一些自己迷茫时的想法，有些东西写下来应该就会稍微好一点；

  • 数据分析：说实话，我现在对于我自己究竟能否胜任数据分析这个职位深表怀疑，而且最近也没有心思去完成优达学院布置的任务，可能数据分析会放弃，重新投入到编程这边；又或者会重新报一个商业数据分析的课程重新入一下门，这个还有待商榷；

  • python 相关：最近一直在忙于后台的开发；包括微信开发的整个授权流程以及和微信后台的一些交互，感觉自己后台方面还是有了较为明显的进步；

  • 数据结构和算法的东西，希望有时间可以开始看看，理解，做题，加深自己；

  • 接下来，每天练习一道 sql 语句，提升自己的 sql 水平，并且试着用 explain 来优化 sql 语句；

    ​

    说一下三月份四月份完成的目标：

    • 看书：《鸟哥的 linux 私房菜》看完；
    • 英语：每日单词 + 每日一篇英文文章；
    • 健身：每周三至四次；
    • 博客：两周一篇保证质量；
    • 数据分析：待定；
    • 数据库相关：每日一道数据库查询题；
    • 算法：leetcode 算法题一道（java、python 两种语言实现）

希望自己接下来能够越来越好。
