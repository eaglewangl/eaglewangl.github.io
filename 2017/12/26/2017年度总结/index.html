2017年度总结

Posted on 2017-12-26 | In 杂谈

年度总结姗姗来迟

时间总是不经意过的飞快，转眼间已经到了17年的年底。还记得年初时许下的承诺，估计现在只能默默伸出我的脸颊挨打了，哎。

<!— more — >

说归说，一年一度的总结私以为还是应该有的，不管怎么样，还是希望一年能比一年强，只有每年都进步一点点，就不怕成功姗姗来迟。

既然是年度总结，套路还是那个套路：首先回顾一下度过的一整年，然后畅想一下未来的新一年，我也不能例外了。

回顾

2017年，可能真的和本命年有关？我也是感觉自己是有点不顺的。年初跳槽找工作就花了将近一个多月的时间，到最后也不能说找到的是自己非常满意的一家公司，只能说还过的去。这个时候已经过了四分之一年，而我感觉这个时候什么都没做，入眼处尽是迷茫不如意。

开始工作后，发现工作中的挑战很大，开始收敛身心准备迎接挑战，刚读懂整个业务，老大突然前往深圳离我们而去，从此公司技术人员只剩下三人，一个 Android（我），一个 iOS，一个产品，后台（老大）突然的离开令我不知所措，当时就是想着学技术所以工资没有要的很高，结果老大在一个半月之后就去了深圳，这个时候还能谈什么学习呢，然后老总找到我，希望我能顶上后台这个任务，于是我开始学习 Python ，学习后台框架，学习如何打印日志，学习如何改后台 bug ，学习 mysql ， 学习用后台的思想看问题而不仅仅是以 Android 的思想看问题。受益匪浅！

于是开始看后台相关的书籍，看 Python 语言，看 Django 框架，看 MySql 入门，看 Redis ， 这大概就是我这一年的最大收获吧，从此结缘后台，走向全栈开发师的行业？哈哈哈哈，我就这么一想，可别当真了，因为我最近又重新开始写 Android 了！- -

吐槽了那么多，说说之前在16年底许下的今年的事情清单：

  • 看书（4/15）：真的是被狠狠扇了一耳光啊，还以为自己是多么有毅力，多么努力，结果一年连15本书的要求都没有做到！
  • 健身（-1）：虽然我在年底的时候办了一张健身卡，然而根本还没有来得及去，理由总是多种多样，但是归结起来就是一个懒，一个怂，怂是因为对健身房里面设备的不熟悉，怕丢人，所以一直没有去，而且还越来越胖了，妈耶，我自己都不敢相信我居然都快成一个胖子了，但是我不很开心，所以我一定会瘦下来的！这是我对自己的考验！
  • 博客（4/20）:原计划是一个月一篇到两篇的产出，结果果然自己是读书读的少啊，很多时候就是有想写的欲望去没有下笔的能量，所以还是要多读书，多思考，结合书中的内容，把学到的东西用自己的语言描述出来，才能真正把知识变成自己的。要努力啊。
  • 英语（60）：算是及格分数吧，也基本上有做到每天15~20分钟的英语单词记忆，语法的话暂时还没有接触到，可能以后会在单词的基础上抽上十分钟每天阅读一篇英文的小文章，来提高自己的英语能力，这可能是当初下的目标里面唯一一个能到及格分数的东西了吧。
  • 新技能（Python）：工作的需要，比大多数人先学了 Python 这门语言，也在思考这门语言之后能让我有什么突出的技能点，暂时看来就两个：一个是做数据分析的工具；一个是网站后台的技能，总之这个还是一个我觉得这一年不错的学习点，接下来也会继续加油。
  • ps：今年报了一个数据分析的课程，吃透之后准备和python结合起来，到时候用python写脚本爬数据，然后用数据分析分析数据，说不定就可以分析一下股票的走势，万一财务自由了岂不是美滋滋！

2018，新的一年，新的挑战。

计划

首先，我对自己的未来依然是迷茫的。听很多人说过，当你迷茫的时候，学习就对了，看书就对了，正如一个好友所说：当答案在是与不是之间，而你不确定的时候，选是就对了，人生一定要积极向上的不是吗，所以我们也要好好学习，生命不息，学习不止。那么接下来就是新一年的计划了：

  • 读书：计划目标-15（不一定技术书籍，可以是文学类，历史类，小说不能算入）
  • 健身：有时间的话一周四次，就算项目再赶也必须保持一周两次的频率，减肥是一件需要持之以恒的事情，虽然唯与爱不可辜负，但是健康也很重要。
  • 博客：保持一月1~2篇，接下来的博客内容可能不单单是 Android 方向的框架代码解析了，可能还会包括数据分析方面的知识，爬虫的知识，后台的一些知识也可能会有，总之希望能保证月出1~2篇。
  • 英语：每天早上15分钟的单词时间，有时间的话加上十分钟的英语文章阅读，阅读和记忆单词时有不熟悉的单词记录到错题本上，晚上睡觉前记得复习。
  • 新技能：1、争取把 Python 精通一点；2、学习部署服务器相关的知识，学习 uwsgi 和 nginx 的知识，以备日后所需；3、有可能的话会看一些机器学习相关的内容，如果看得懂就看，实在不是那块料就果断尝试别的东西。
  • 女朋友？？？大哥你想什么呢，是学习不吸引人吗？是书里面没有颜如玉吗？你一天天脑子里面装的都是些啥！

到这里，总结也就结束了，虽然是本命年，虽然有点不顺，但是有很多东西还是值得记录的。

很幸运 2017 遇到了这么多可爱的你们，很遗憾 2017 也带走了我的一些亲人，但是时光啊，永远不会停步，希望未来，我能和你们一样变得更加美好。

完。
